# 🌤️ dk Weather website

A modern and responsive **Weather Forecast Application** built using **React.js** and **OpenWeather API**.  
Get real-time weather updates and a 5-day forecast for any city worldwide 🌍

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Real-time temperature, humidity, pressure, and visibility  
- 🌅 Sunrise & 🌇 Sunset timings   
- 📅 5-Day weather forecast  
- 🎨 Clean neumorphic UI design  
- ⚡ Fast and responsive (mobile-friendly)  
- ❌ Error handling (invalid city / API issues)  
- ⏳ Loading states for better UX  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **API:** OpenWeather API  
- **HTTP Client:** Axios  
- **Icons:** React Icons  

---
## 🌐 API Used

- Current Weather API  
- 5 Day / 3 Hour Forecast API  

## 🧠 Key Learnings

- Working with real-world APIs  
- Handling async data in React  
- Managing loading & error states  
- Forecast data transformation  
- Timezone-based date & time handling
